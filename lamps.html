Lamps

visit the website. a web design & development project.

Role
-

  • 1 identity design
  • 2 HTML / CSS / JS
  • 3 home, cart, about, rating page

Context
-

February 2018

This is my first time coding a website that implements design and functionality together. Within 4 weeks, my partner and I were able to go through an initial style guide, a site map, and an additive process of iteration to completion.

Role
-

I created the visual identity of the website, starting with the home page menu, then the grid structure for the rest of the page. I also designed the cart, about, rating pages and the footer, which follow grid structures that can be found on the home page. This includes responsiveness and functionality of all buttons, hover states, and JavaScript of the collapsible menu.

I also did the style guide to hold all of the visual elements for the website and a quick citations page. You can check the Github for more specific details on contributions.

Take-aways
-

Naming conventions are useful for those who did not create the document. Although I created most of the class and ID names, they could have been named more clearly for my partner to iterate and work with as he started his pages after me and had to follow my naming conventions. I did not receive any complaints about naming, but I think that it would have been a much quicker process for my partner if it was more intuitive.
