Lamps

visit the website. a web design & development project.

Role
-

  • 1 identity design
  • 2 HTML/CSS/JS
  • 3 home, cart, about, rating page

Goal
-

To create a responsive website including a home, multi-product, single-product, check-out pages, and a style guide appropriate for your fictional company's audience.

Tools
-

  • Atom text editor
  • GitKraken
  • Axure RP
  • Illustrator/Photoshop

Context
-

February 2018
Course - IAT 339 web design and development

This is my first time I coded a website designed for a wide variety of users. Within 4 weeks, my partner and I established an initial prototype using Axure RP, a style guide, a site map, and proceeded through an additive process of iteration.

Role
-

I created the visual identity of the website, starting with the home page, menu, then the grid structure. I also designed the cart, about, rating pages and the footer, which includes responsiveness and functionality of all buttons, hover states, and JavaScript of the collapsible menu.

I also created a style guide for branding guidelines and a citations page. You can check the Github for more specific details on contributions.


GitHub

Audience
-

A broad range of users from renovators to business owners pushed forward a visual language that increases the trustworthiness of the site itself. The high-end lighting photos shown on the website should compliment the aesthetic structure of the website, which is accomplished through a clear grid structure, pale colours, and clear type.

Take-aways
-

Naming conventions used in this project should have been more intuitive for my partner to work with. This resulted in many of his own classes being created, as he was unable to understand many functions of the existing code.

Thinking about how the html is structured is an area that I struggled to organize my thoughts in. I overcame this by sketching and creating the grid structure first.
